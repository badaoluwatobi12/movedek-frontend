# SendAm Frontend

React + Vite + TypeScript frontend for SendAm.

## Main folders

- `src/pages` — public, customer, courier, merchant, admin screens
- `src/components` — layout, common, delivery, courier, merchant, admin components
- `src/services` — frontend service layer
- `src/hooks` — reusable hooks
- `src/context` — app context providers
- `src/lib` — constants, formatters, pricing, storage, validators
- `src/types` — frontend type exports

## Run

```bash
npm install
npm run dev
```
